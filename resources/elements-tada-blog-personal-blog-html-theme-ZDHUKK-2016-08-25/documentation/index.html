SECTION: Content & Sidebar

Section tag <!section class="tada-container content-posts"> include the content of the page/post: content text and sidebar (where is included)

  1. content: <!div class="content"> inside this tag is include all content text
  2. sidebar: <!div class="sidebar"> inside this tag is include all sidebar widget

Article

Each article is comment by this comment tag <!-- ARTICLE 1 -->
where the numeber is a number of article. Also each article is between two specific tag <article> and </article>

Article is organizated in this DIV:

  1. post-image: <div class="post-image"> </div> where included image post (tag IMG) and category (class category)
  2. post-text: <div class="post-text"> </div> where included date post (span with class date), title post (H2) and text post (p with class text)
  3. post-info: <div class="post-info"> </div> where included author post (div with class post-by) and social share (div with class extra-info)

Footer

Footer area is included in tag <!--footer--> Footer is organizated in:

  1. widget-gallery: widget instagram/gallery uses a class="widget widget-gallery" and included all image gallery using IMG tag for each image
  2. footer-bottom: is area for footer bottom where you can put a copyright text

General Faq

  1. How to disable Slider
    Very easy you need to find <!-- SLIDER --> and delete all HTML code until <!-- #SLIDER -->
  2. How to disable sticky menu
    find this menu-sticky and replace with no-menu-sticky
  3. How to remove image background to body
    open style.css find and remove this: background-image:url('../img/bg.jpg');
  4. How to remove preloader
    Very easy find and remove it <div id="preloader-container"> <div id="preloader-wrap"> <div id="preloader"></div> </div> </div>
    or find PRELOADER comment
  5. How to remove sidebar in page where the page sidebar is included.
    find class sidebar or comment <!-- SIDEBAR -->on the page and remove all code until <!-- #SIDEBAR -->
    Now find this class col-xs-8 and replace with col-xs-12
  6. How to add sidebar in page where the page sidebar is not included.
    find class content or comment <!-- CONTENT --> and find the class col-xs-12 and replace with col-xs-8
    now add this code before it <div class="clearfix"></div>

    <!-- SIDEBAR -->
    <div class="sidebar col-xs-4">put here your content sidebar</div>
  7. How to change color
    open style.css find this [0. Color Skin] and now you need to change all color in relative classes.
    For example main color is #f6f6f6 and where do you can see this color you need to change it with your favorite color.
    In CSS Table of contents (inside this css file) you can see all color used in our template.
